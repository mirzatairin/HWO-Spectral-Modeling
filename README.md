# HWO-Sim: Gaussian Modeling for Exoplanet Biosignature Detectability

### Project Overview
This research tool was developed to support the mission goals of NASA’s **Habitable Worlds Observatory (HWO)**. It simulates how we might detect life-sustaining molecules in the atmospheres of distant planets.

### Engineering Features
* **Gaussian Synthesis:** Accurate modeling of molecular absorption lines (H2O, CO2, CH4).
* **Signal Processing:** Peak detection utilizing prominence thresholds to filter instrument noise.
* **Metric Calculation:** Automated Habitability Index based on biosignature detection.

### Technical Stack
* **Language:** Python 3.x
* **Libraries:** NumPy, SciPy (Signal Processing), Matplotlib (Visualization)
