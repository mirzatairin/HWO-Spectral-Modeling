# HWO-Sim: Gaussian Modeling for Exoplanet Biosignature Detectability
### Exoplanet Atmosphere Spectral Analyzer

### Project Overview
This project simulates and analyzes exoplanet atmospheric spectra to identify molecular absorption features and assess potential habitability. It simulates how we might detect life-sustaining molecules in the atmospheres of distant planets.

### Engineering Features
* **Gaussian Synthesis:** Accurate modeling of molecular absorption lines (H2O, CO2, CH4).
* **Signal Processing:** Peak detection utilizing prominence thresholds to filter instrument noise.
* **Metric Calculation:** Automated Habitability Index based on biosignature detection.

### Technical Stack
* **Language:** Python 3.x
* **Libraries:** NumPy, SciPy (Signal Processing), Matplotlib (Visualization)

## Motivation
Inspired by exoplanet spectroscopy research, this project demonstrates how computational methods can support the detection of atmospheric biomarkers.
